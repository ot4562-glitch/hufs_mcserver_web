06

설치 후 과정

모드 설치 완료 후 아래 순서대로 진행하세요.

1

런처 프로필 선택

마인크래프트 런처에서 Fabric Loader 프로필을 선택하고 실행하세요.

2

단축키 설정

메인화면 → 설정 → 조작 → 키 지정에서 붉은색으로 표시된 중복 키를 수정하세요.

3

비디오 설정

메인화면 → 설정 → 비디오 설정에서 전체 설정을 확인하세요.
쉐이더 팩 설정에서 반드시 켜주세요.

4

리소스팩 활성화

설정 → 리소스 팩에서 사용 가능 목록의 팩을 선택됨으로 이동하세요.
