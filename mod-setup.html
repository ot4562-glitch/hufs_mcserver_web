Section 6: 설치 후 과정

1

런처 프로필 선택

마인크래프트 런처에서 Fabric Loader 프로필을 선택하고 실행하세요.

2

단축키 설정

메인화면 → 설정 → 조작 → 키 지정에서 붉은색으로 표시된 중복 키를 수정하세요.

3

비디오 설정

메인화면 → 설정 → 비디오 설정에서 왼쪽 스크롤을 내리며 전체 설정을 확인하세요. PC 사양과 취향에 맞춰 조정하시면 됩니다. 쉐이더 팩 설정에서 다운로드한 것을 반드시 켜주세요.

4

리소스팩 활성화

메인화면 → 설정 → 리소스 팩에서 다운로드한 팩을 왼쪽 사용 가능 목록에서 오른쪽 선택됨으로 이동하세요.
